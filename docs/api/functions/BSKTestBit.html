BSKTestBit

Module: BSKBitSet
BSKBOOL BSKTestBit( BSKBitSet* set, BSKUI32 bit );
Inputs:
  • set - the set containing the bit to test.
  • bit - the index of the bit to test.
Returns: BSKTRUE if the indicated bit is set, or BSKFALSE otherwise.

Description: Returns BSKTRUE if the indicated bit is set, or BSKFALSE if it is not, where bit is the 'index' of the bit to test.
