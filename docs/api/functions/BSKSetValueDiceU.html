BSKSetValueDiceU

Module: BSKValue
void BSKSetValueDiceU( BSKValue* value, 
                       BSKI16  count, 
                       BSKUI16 type,
                       BSKCHAR op,
                       BSKI16  modifier,
                       BSKUI32 units );
Inputs:
  • value - the value to set the dice value in.
  • count - the number of dice (if negative, the dice will always result in a negative number).
  • type - the type of the dice (sides per die).
  • op - the operator for the modifier ('+' or '*').
  • modifier - the modifier for the dice computation.
  • units - the identifier of the units to use (or 0 to use no units).

Description: Same as BSKSetValueDice but also sets the units (the VF_UNITS flag is set).
