BSKCategoryIntersection

Module: BSKCategory
BSKCategory* BSKCategoryIntersection( BSKCategory* first, 
                                      BSKCategory* second );
Inputs:
  • first - the first category
  • second - the second category
Returns: a new category containing the intersection of the two parameters.

Description: Creates a new category that is the intersection of the given 'first' and 'second' categories -- in other words, it contains only the items that exist in both the 'first' and 'second' categories.
