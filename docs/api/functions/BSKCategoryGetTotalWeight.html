BSKCategoryGetTotalWeight

Module: BSKCategory
BSKUI32 BSKCategoryGetTotalWeight( BSKCategory* category );
Inputs:
  • category - the category object to get the total weight of.
Returns: the total weight of the given category.

Description: Obtains the total weight of the category, which is the sum of all the weights of the category's members.
