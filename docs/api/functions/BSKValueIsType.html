BSKValueIsType

Module: BSKValue
BSKBOOL BSKValueIsType( BSKValue* value, BSKUI16 type );
Inputs:
  • value - the value to check the type of.
  • type - the type (or combination of types) to test.
Returns: BSKTRUE if the value is one of the given types, BSKFALSE otherwise.

Description: Checks the type of the given value against the types given. The types may be one or more of the VT_xxx constants.
