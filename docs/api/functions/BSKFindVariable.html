BSKFindVariable

Module: BSKVariable
BSKVariable* BSKFindVariable( BSKVariable* list, BSKUI32 id );
Inputs:
  • list - the list to search.
  • id - the id of the variable to return.
Returns: the requested BSKVariable object, or 0 if one can't be found with the given identifier.

Description: Searches the given list for a variable with the given identifier.
