BSKUnitGetReference

Module: BSKUnitDefinition
BSKUI32 BSKUnitGetReference( BSKUnitDefinition* unit );
Inputs:
  • unit - the unit definition to get the "reference" unit of.
Returns: the reference unit of the given unit definition.

Description: Obtains the reference unit of the given unit definition. The reference unit is the unit that the given unit definition is defined relative to (ie, "pounds" may be defined relative to "ounces", in which case "ounces" is the reference unit).
