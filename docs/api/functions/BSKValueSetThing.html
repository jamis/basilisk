BSKValueSetThing

Module: BSKValue
void BSKValueSetThing( BSKValue* value, BSKThing* thing );
Inputs:
  • value - the value object.
  • thing - the thing to set as the data.

Description: Makes the given value of type VT_THING and puts the given thing as it's data.
