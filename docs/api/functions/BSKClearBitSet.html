BSKClearBitSet

Module: BSKBitSet
BSKBitSet* BSKClearBitSet( BSKBitSet* set );
Inputs:
  • set - the set to clear.
Returns: the 'set' parameter.

Description: Initializes the given bit set to 0, thus clearing all bits in the set.
