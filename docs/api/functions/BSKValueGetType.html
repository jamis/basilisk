BSKValueGetType

Module: BSKValue
BSKUI16 BSKValueGetType( BSKValue* value );
Inputs:
  • value - the value to get the type of.
Returns: the type of the value.

Description: Obtains the type of the value. This will be one (or possibly a combination) of the VT_xxx constants.
