BSKCopyBitSet

Module: BSKBitSet
BSKBitSet* BSKCopyBitSet( BSKBitSet* dest, BSKBitSet* src );
Inputs:
  • dest - the set to recieve the contents of 'src'.
  • src - the set to copy.
Returns: the 'dest' set.

Description: Copies the set 'src' into the set 'dest'.
