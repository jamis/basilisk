BSKStackFrameGetParameters

Module: BSKExec
BSKValue** BSKStackFrameGetParameters( BSKStackFrame* frame );
Inputs:
  • frame - the stack frame to query.
Returns: An array of BSKValue pointers that are the parameters that were passed to the given stack frame.

Description: Obtains the array of BSKValue pointers that are the parameters to the given stack frame.
