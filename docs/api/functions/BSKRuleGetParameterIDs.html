BSKRuleGetParameterIDs

Module: BSKRule
BSKUI32* BSKRuleGetParameterIDs( BSKRule* rule );
Inputs:
  • rule - the rule to get the parameter list of.
Returns: the array of id's of the declared parameters to the rule.

Description: Obtains the ID's of the rule's declared parameters, in the order in which they were declared.
