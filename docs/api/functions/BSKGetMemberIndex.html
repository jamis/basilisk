BSKGetMemberIndex

Module: BSKCategory
BSKI32 BSKGetMemberIndex( BSKCategory* category, BSKCategoryMember* member );
Inputs:
  • category - the category containing the member.
  • member - the member to obtain the index of.
Returns: the index of the member in the category, or -1 if the category does not contain that member.

Description: Returns the index of the requested member in the given category, or -1 if the member does not exist in the category.
