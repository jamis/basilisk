BSKClearBit

Module: BSKBitSet
BSKBitSet* BSKClearBit( BSKBitSet* set, BSKUI32 bit );
Inputs:
  • set - the set containing the bit to clear.
  • bit - the index of the bit to clear (0-based).
Returns: The 'set' parameter.

Description: Clears the indicated bit in the set, where bit is the index of the bit to clear.
