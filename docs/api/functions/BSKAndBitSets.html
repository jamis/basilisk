BSKAndBitSets

Module: BSKBitSet
BSKBitSet* BSKAndBitSets( BSKBitSet* dest, BSKBitSet* first, BSKBitSet* second );
Inputs:
  • dest - the bitset that will recieve the result of the operation. This may be the same set as either 'first' or 'second'.
  • first - the first bit set (may be the same as 'second').
  • second - the second bit set.
Returns: The 'dest' bitset.

Description: Combines sets 'first' and 'second' into 'dest', using a bitwise AND operation.
