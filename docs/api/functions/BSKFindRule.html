BSKFindRule

Module: BSKRule
BSKRule* BSKFindRule( BSKRule* list, BSKUI32 id );
Inputs:
  • list - the list of rules to search.
  • id - the id of the rule to find.
Returns: the rule with the given id, or 0 if the rule is not found.

Description: Looks for the given rule in the given list of rules.
