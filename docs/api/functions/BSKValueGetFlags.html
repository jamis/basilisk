BSKValueGetFlags

Module: BSKValue
BSKUI8 BSKValueGetFlags( BSKValue* value );
Inputs:
  • value - the value to get the flags of.
Returns: the flags for the given value.

Description: Obtains the flags for the given value. These will be a combination of the VF_xxx constants, or 0.
