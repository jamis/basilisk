BSKValueSetCategory

Module: BSKValue
void BSKValueSetCategory( BSKValue* value, BSKCategory* cat );
Inputs:
  • value - the value object
  • cat - the category to set in the value.

Description: Makes the given value of type VT_CATEGORY and sets the given category as it's data.
