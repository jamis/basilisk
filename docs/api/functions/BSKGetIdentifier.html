BSKGetIdentifier

Module: BSKIdentifierTable
void BSKGetIdentifier( BSKIdentifierTable* table,
                       BSKUI32 id,
                       BSKCHAR* buffer,
                       BSKUI32 length );
Inputs:
  • table - the table to query for a given identifier.
  • id - the identifier to retrieve.
  • buffer - the buffer to copy the string into.
  • length - the size (in bytes) of the buffer.

Description: Retrieves the requested identifier into the given buffer, but copies no more than 'length' bytes (and always null-terminates the buffer).
