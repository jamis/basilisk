BSKFindUnitDef

Module: BSKUnitDefinition
BSKUnitDefinition* BSKFindUnitDef( BSKUnitDefinition* list,
                                   BSKUI32 id );
Inputs:
  • list - the list to search.
  • id - the id of the unit definition to return.
Returns: the requested unit definition, or 0 if one with the given id could not be found.

Description: Looks for the requested unit definition in the list.
