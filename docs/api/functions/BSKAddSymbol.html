BSKAddSymbol

Module: BSKSymbolTable
void BSKAddSymbol( BSKSymbolTable* table, 
                   BSKUI8 type, 
                   BSKUI8 flags, 
                   BSKUI32 id );
Inputs:
  • table - the symbol table to add to.
  • type - the type of the symbol (one of the ST_xxx constants).
  • flags - flags that further define the symbol (a combination of the SF_xxx constants).
  • id - the identifier that the symbol describes.

Description: Adds a new symbol to the given table, of the given type, with the given flags, and with the given identifier. This routine does not check to see if the symbol already exists in the table.
