BSKOrBitSets

Module: BSKBitSet
BSKBitSet* BSKOrBitSets( BSKBitSet* dest, BSKBitSet* first, BSKBitSet* second );
Inputs:
  • dest - the bitset that will contain the result (may be the same as either 'first' or 'second').
  • first - the first bitset.
  • second - the second bitset.
Returns: The 'dest' set.

Description: Combines sets 'first' and 'second' into 'dest', using a bitwise OR operation.
