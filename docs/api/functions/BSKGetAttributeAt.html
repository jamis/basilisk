BSKGetAttributeAt

Module: BSKThing
BSKAttribute* BSKGetAttributeAt( BSKThing* thing, BSKUI32 idx );
Inputs:
  • thing - the thing containing the attribute to return.
  • idx - the index of the attribute to return.
Returns: returns the attribute at the given index in the thing's attribute list, or 0 if there is no attribute at the given index.

Description: Gets the attribute at the given index in the thing's attribute list.
