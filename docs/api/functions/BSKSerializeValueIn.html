BSKSerializeValueIn

Module: BSKValue
BSKI32 BSKSerializeValueIn( struct __bskdatabase__* db, BSKValue* value, BSKStream* stream );
Inputs:
  • db - the database used to lookup referenced entities.
  • value - the value to recieve the value on the stream.
  • stream - the stream containing the serialized value.
Returns: 0 if successful.

Description: Reads a new value from the stream into the 'value' parameter, using the database object as necessary to look up database members (things, categories, and rules).
