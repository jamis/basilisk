BSKSetValueNumberU

Module: BSKValue
void BSKSetValueNumberU( BSKValue* value, BSKFLOAT number, BSKUI32 units );
Inputs:
  • value - the value to set the number in.
  • number - the number to set.
  • units the identifier of the units to associate with the number, or 0 to not associate any units with the number.

Description: Same as BSKSetValueNumber, but also sets the units of the number (the VF_UNITS flag is set, in this case).
