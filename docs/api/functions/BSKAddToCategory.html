BSKAddToCategory

Module: BSKCategory
void BSKAddToCategory( BSKCategory* category, 
                       BSKUI16 weight, 
                       BSKNOTYPE newMember );
Inputs:
  • category - the category to add the item to.
  • weight - the integer weight of the new item, which determines its likelihood of being randomly chosen relative to the other items in the category. (See BSKCategory).
  • newMember - the new member being added. This must be either null (0), or a pointer to a BSKThing, BSKRule, or BSKCategory object.

Description: Adds the given member to the given category with the given weight. 'newMember' must either be null (0), or it must point to a BSKThing, BSKRule, or BSKCategory object. The object is added to the END of the category contents list, contrary to the behavior of most other lists in Basilisk (which add the new item to the front of the list).
