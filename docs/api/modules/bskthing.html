BSKThing

Defined in: bskthing.h

A BSKThing object is what makes the whole Basilisk universe go round. It represents a single entity in the database, and possesses a set of attributes that describe it.

A thing's attributes may be strings, numbers (including dice), booleans, other things, categories, and even rules.
